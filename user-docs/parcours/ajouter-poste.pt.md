# Adicionar um posto de trabalho

<span class="role-badge">Admin funcional</span>

Crie um posto de trabalho e associe-o a uma operação numa linha.

## 1. Abrir os postos

Menu **Administração → Postos de trabalho**, depois **Novo posto**.

<figure class="screenshot" markdown>
![Lista dos postos](../assets/screenshots/pt/poste-liste.png)
<figcaption>Postos de trabalho</figcaption>
</figure>

## 2. Configurar o posto

| Campo | Função |
|-------|------|
| **Nome** | Designação do posto (o **código** é gerado automaticamente) |
| **Linha de produção** | A linha do posto, ou **Todas** para um posto partilhado |
| **Operação** | A operação gerida — a lista depende da linha escolhida |
| **Máquina pilotada** | *(opcional)* máquina associada |

<figure class="screenshot" markdown>
![Formulário do posto](../assets/screenshots/pt/poste-form.png)
<figcaption>Configuração do posto</figcaption>
</figure>

!!! info "Linha e depois operação"
    Escolha primeiro a **linha**: a lista de **operações** limita-se então
    às configuradas nessa linha. «Todas as linhas» propõe as
    operações comuns (posto partilhado).

!!! tip "Máquina pilotada"
    Indicar uma máquina ativa o botão **Descarregar a receita** no
    terminal e transmite a receita à máquina.

## 3. Guardar

Toque em **Guardar**: o posto fica pronto a ser selecionado no início de sessão
por crachá.

![Posto criado](../assets/screenshots/pt/poste-cree.png)
<figcaption>Posto adicionado</figcaption>
