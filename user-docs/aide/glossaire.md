# Glossaire

| Terme | Définition |
|-------|------------|
| **OF — Ordre de fabrication** | Commande de production pour un modèle/coloris, avec quantités par taille. Importé de l'ERP ou créé manuellement. |
| **Sous-OF** | Subdivision d'un OF en lots plus petits (souvent une taille par lot) pour un suivi granulaire. Matérialisé par un bac. |
| **Bac** | Contenant physique d'un sous-OF, identifié par un QR code scanné sur le terminal. |
| **Opération** | Étape du processus de fabrication (coupe, piqûre, montage…). Ordonnée par une séquence. |
| **Pointage** | Déclaration du début et de la fin d'une opération sur un sous-OF. |
| **Ligne de production** | Regroupement d'opérations par zone physique, pour suivre la production par ligne. |
| **Poste de travail** | Emplacement physique associé à une opération (et éventuellement une équipe). |
| **Rebut** | Pièce non conforme. Type **P** (paire), **G** (gauche) ou **D** (droite). |
| **Réintégration** | Reconstitution d'une paire à partir de chaussures orphelines valides du stock. |
| **Stock de réintégration** | Chaussures individuelles (G ou D) valides, en attente d'être appariées. |
| **Orpheline** | Chaussure valide dont l'opposée a été rebutée, placée au stock de réintégration. |
| **Contrôle qualité (QC)** | Vérification de conformité après certaines opérations, selon un pourcentage de contrôle. |
| **Efficacité** | Ratio (sortie / entrée) × 100 d'une opération, ligne ou opérateur. |
| **Équipe** | Plage horaire de travail (matin 6 h–14 h, après-midi 14 h–22 h, nuit 22 h–6 h par défaut). |
| **ERP / PRIOS** | Système de gestion source des OF, transmis via fichiers XML dans un dossier surveillé. |
