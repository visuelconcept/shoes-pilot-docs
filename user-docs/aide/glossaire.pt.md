# Glossário

| Termo | Definição |
|-------|------------|
| **OF — Ordem de fabrico** | Encomenda de produção para um modelo/cor, com quantidades por tamanho. Importada do ERP ou criada manualmente. |
| **Sub-OF** | Subdivisão de uma OF em lotes mais pequenos (frequentemente um tamanho por lote) para um acompanhamento granular. Materializada por um contentor. |
| **Contentor** | Recipiente físico de uma sub-OF, identificado por um código QR lido no terminal. |
| **Operação** | Etapa do processo de fabrico (corte, costura, montagem…). Ordenada por uma sequência. |
| **Registo** | Declaração do início e do fim de uma operação numa sub-OF. |
| **Linha de produção** | Agrupamento de operações por zona física, para acompanhar a produção por linha. |
| **Posto de trabalho** | Localização física associada a uma operação (e eventualmente a uma equipa). |
| **Rejeição** | Peça não conforme. Tipo **P** (par), **G** (esquerdo) ou **D** (direito). |
| **Reintegração** | Reconstituição de um par a partir de sapatos órfãos válidos do stock. |
| **Stock de reintegração** | Sapatos individuais (G ou D) válidos, a aguardar emparelhamento. |
| **Sapato órfão** | Sapato válido cujo oposto foi rejeitado, colocado no stock de reintegração. |
| **Controlo de qualidade (QC)** | Verificação de conformidade após certas operações, segundo uma percentagem de controlo. |
| **Eficiência** | Rácio (saída / entrada) × 100 de uma operação, linha ou operador. |
| **Equipa** | Período horário de trabalho (manhã 6 h–14 h, tarde 14 h–22 h, noite 22 h–6 h por predefinição). |
| **ERP / PRIOS** | Sistema de gestão de origem das OF, transmitidas através de ficheiros XML numa pasta monitorizada. |
