# Adicionar um operador

<span class="role-badge">Supervisor</span>

Crie um operador para que possa iniciar sessão no terminal por crachá.

## 1. Abrir a lista

Menu **Administração → Operadores**, depois toque em **Novo operador**.

<figure class="screenshot" markdown>
![Lista dos operadores](../assets/screenshots/pt/operateur-liste.png)
<figcaption>Lista dos operadores</figcaption>
</figure>

## 2. Preencher o operador

Introduza os três campos obrigatórios:

| Campo | Exemplo |
|-------|---------|
| **Crachá** | `12345678` |
| **Nome próprio** | Jean |
| **Apelido** | Dupont |

<figure class="screenshot" markdown>
![Formulário do operador](../assets/screenshots/pt/operateur-form.png)
<figcaption>Criação de um operador</figcaption>
</figure>

!!! info "O crachá = o início de sessão"
    O número do crachá é o identificador lido no terminal. Tem de ser
    **único**.

## 3. Guardar

Toque em **Guardar**: o operador aparece na lista e já pode iniciar
sessão.

<figure class="screenshot" markdown>
![Operador criado](../assets/screenshots/pt/operateur-cree.png)
<figcaption>Operador adicionado à lista</figcaption>
</figure>

!!! tip "Desativar um operador"
    Um operador nunca é eliminado: é **desativado** (filtro Ativos /
    Inativos). Reative-o quando necessário com um novo crachá.
