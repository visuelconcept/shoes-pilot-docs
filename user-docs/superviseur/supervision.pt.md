# Supervisionar a produção

<span class="role-badge">Supervisor</span> <span class="role-badge">Admin</span>

Acompanhe a produção em tempo real. Os painéis atualizam-se sozinhos (15–30 s) e
analisam-se segundo vários eixos.

Em curso, concluídos do dia, pares produzidos, sucatas e a produção hora a hora.
